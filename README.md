# Write Deep Writer

自适应深度写作智能体，将任意输入转化为连贯、深度的文字内容。

## 概述

该智能体将口语化或碎片化内容（视频采访、演讲、播客）转化为结构严谨、内容深度的文字。它能自动：
- 从输入中识别写作原则、目标受众和场景
- 处理口语化内容并去除冗余
- 合并相似观点以增强清晰度
- 优化逻辑同时保持原有结构
- **通过联网搜索获取真实案例和数据**
- 提供三阶段工作流程并跟踪进度

## 快速开始

```
输入：任意内容（视频采访文本、演讲稿、文章、播客文本）

输出：深度、结构化的文字内容（5000-10000字）
```

## 核心能力

### 九层处理系统

1. **需求识别** - 从输入中提取写作原则、受众、场景和目的
2. **Brief 文档构建** - 构建写作基准文档，包含创作者人设、受众画像和立场
3. **预处理** - 识别并处理口语化内容、重复和碎片化观点
4. **观点提取与聚类** - 提取并按语义相似度合并相似观点
5. **逻辑分析** - 识别因果、并列、递进、对比和总结关系
6. **结构优化** - 保持核心框架的同时增强逻辑并删除重复
7. **联网研究** - 搜索真实案例、产品、数据和研究文献
8. **深度撰写** - 补充背景知识、数据、示例并优化表达
9. **质量检查** - 验证所有约束条件后再输出最终内容

## 工作流程

### 三阶段流程（带任务跟踪）

```
阶段一：分析与规划
  - 需求识别
  - Brief 文档构建
  - 预处理
  - 观点提取与聚类
  - 逻辑分析

阶段二：结构设计
  - 结构优化

阶段三：内容撰写
  - 联网研究（获取案例和数据）
  - 深度撰写
  - 质量检查
```

## 联网搜索功能

该智能体内置联网搜索能力，可自动获取：
- 真实的产品案例和公司信息
- 最新的研究报告和统计数据
- 专家观点和政策文件
- 可信来源的引用

### 无需额外配置

在支持的环境中（如 Claude Code），联网搜索是**内置功能**，无需额外安装或配置 API 密钥。

智能体会自动：
1. 识别各章节的研究需求
2. 生成针对性搜索查询
3. 检索相关信息
4. 整合到深度写作中
5. 添加来源引用

---

### 外部搜索 API 推荐

如果您希望使用外部搜索服务而非内置功能，我们推荐使用 **Anspire Open API**。

#### 获取 Anspire Open API Key

1. 访问 Anspire 网站并注册账户
2. 进入 API 设置或开发者控制台
3. 生成新的 API 密钥
4. 复制 API 密钥用于配置

#### 配置方法

```yaml
# 配置示例
web_search:
  provider: anspire
  api_key: "YOUR_ANSPIRE_API_KEY"
  endpoint: "https://api.anspire.com/v1/search"
```

#### 使用 Anspire Open API 的优势

- **高质量搜索结果**，支持高级过滤
- **稳定可靠**，响应速度快
- **灵活定价**，满足不同使用需求
- **开发者友好**，提供完善的文档

更多详情请访问 Anspire Open API 官方文档。

## 使用方法

使用该智能体时，提供任意输入内容并遵循三阶段工作流程。每个阶段都需要用户确认后才能继续。
